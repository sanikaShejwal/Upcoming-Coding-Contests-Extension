# Upcoming Contests Extension

A simple and lightweight Chrome Extension that displays upcoming coding contests from multiple competitive programming platforms using the Clist API. Built with HTML, JavaScript, and Bootstrap.

## 📸 Screenshot

![Upcoming Contests Extension](screenshots/preview.png)

## ✨ Features

* Displays upcoming coding contests for the next 7 days
* Fetches real-time contest data using the Clist API
* Clean and responsive user interface with Bootstrap
* Easy to install and use as a Chrome Extension
* Supports contests from multiple coding platforms

## 🛠️ Technologies Used

* HTML5
* JavaScript (ES6)
* Bootstrap 5
* Clist API
* Chrome Extension Manifest

## 🚀 Installation

1. Clone this repository:

```bash
git clone https://github.com/your-username/upcoming-contests-extension.git
```

2. Open Google Chrome and navigate to:

```text
chrome://extensions/
```

3. Enable **Developer Mode**.

4. Click **Load Unpacked**.

5. Select the project folder.

6. Click the extension icon in the toolbar to view upcoming contests.

## 📌 How It Works

* The extension sends requests to the Clist API.
* Contest information is fetched dynamically.
* Upcoming contests scheduled within the next 7 days are displayed in the popup window.
* Users can quickly check contest schedules without visiting multiple websites.
